## 📁 Projects Overview

<details>
<summary><b>🧠 Compiler — Click to Expand</b></summary>

<br/>

> A full or partial **compiler pipeline** implemented in C/C++, covering the theoretical and practical stages of language processing.

### Stages Implemented

- **Lexical Analysis** — Tokenizer/scanner consuming raw source code and emitting token streams
- **Syntax Analysis** — Parser constructing Abstract Syntax Trees (AST) from token streams
- **Semantic Analysis** — Type checking, scope resolution, and symbol table management
- **Code Generation** — Intermediate representation (IR) or direct target code emission

### Concepts Demonstrated

- Finite Automata & Regular Expressions (Lexer)
- Context-Free Grammars & Parse Trees (Parser)
- Three-Address Code / Intermediate Representation
- Error recovery and diagnostic reporting

**Languages of Target / Implementation:** C, C++, Makefile, CMake

</details>

<details>
<summary><b>🎮 Games — Click to Expand</b></summary>

<br/>

> Fully playable **game implementations** using C++ demonstrating OOP design patterns, game loops, collision detection, AI logic, and rendering pipelines.

### Possible / Planned Projects

- **Classic Arcade Games** — Snake, Tetris, Pong (terminal or SDL2)
- **AI-Driven Games** — Minimax / Alpha-Beta pruning for 2-player strategy games
- **Physics Simulations** — Particle systems, gravity, collision engines
- **Puzzle Games** — Grid-based logic puzzles with solver algorithms

### Concepts Demonstrated

- Game loop architecture (fixed timestep)
- 2D rendering with SDL2 / ncurses
- Input handling, collision detection, state machines
- AI opponents using search algorithms

</details>

## 🚀 Getting Started

### Prerequisites

```bash
# Install GCC/G++ and CMake (Ubuntu/Debian)
sudo apt update && sudo apt install -y g++ cmake make git

# Verify installations
g++ --version
cmake --version
```

### Clone & Build

```bash
# Clone the repository
git clone https://github.com/ik-awais/Projects.git
cd Projects

# Navigate to a specific project
cd compiler   # or: cd games/<project-name>

# Build using CMake
mkdir build && cd build
cmake ..
make

# Or with Makefile directly
make
```

### Run a Project

```bash
# Example: Run compiler
./compiler <source_file>

# Example: Run a game
./game
```
